## How to run it (with your own mongodb server) 
### For back-end
please install nodemon first.
```
cd back-end
npm install
(you have to create a config.js which contains the mongodb conenction path and private key for jwtï¼‰
nodemon server.js
```



### For front-end
```
cd front-end
npm install
ng s -o
```
